Black Desert

Description

펄어비스의 대표 게임인 MMORPG 검은 사막의 컨텐츠 기획 및 게임 제작 작업에 참여했다. 프론트 부분의 스트립트 부분이나 백엔드 부분의 기능 구현을 같이 맡았다. js와 Lua를 사용하여 프론트부분을 제작했고, java 를 사용하여 백엔드 기능을 구현하였다.

검은 사막 내의 그림자 전장 관련 컨텐츠와 Jira를 통한 300개의 크고 작은 버그 및 이슈를 수정하고, 생활 컨텐츠 및 길드전에 관련된 작업을 많이 했다.

회사에 다니면서 크게 불편했던 점이 있는데, 다른 분께 바인딩 함수 사용 방법을 받으면 오작동을 일으키는 부분이 꽤 많았다. 그래서 이 부분에 대하여 계속 물어보러 다녀야 했고(Slack 을 사용하여 커뮤니케이션을 했지만 씹거나 자리 비울때가 많았다 --;), 시간적인 손해가 컸다. 바인딩 함수에 대한 코드 열람 권한이 부여되었다면 편했겠지만, 경력있는 사람만 코드 열람이 가능했었다.

또한, 신입들이 의견 제시를 하면 묵인 되는 경우가 많았다. 팀장급 의견이 최우선되어서 기계처럼 일만 했었던 것 같다.

사용한 언어는 java, js , xml , Lua를 사용했고 펄어비스 자체 제작 엔진인 오션 엔진을 사용하여 개발하였다.
